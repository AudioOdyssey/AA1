Navigating Treeview


1) Select starting location:

Utilize the drop down menu to select the location you would like to view first

2) Use decision buttons to navigate

You may select any of the available decisions to refresh the page with the subsequent location as the starting location and see the next decision. You may repeat this step to see the whole story or at anytime you may select a new starting location.

Note:

If no location link appears than that decision leads NOWHERE. Use the "edit decision" links to edit the decisions for a particular location. For more info visit our story help page
